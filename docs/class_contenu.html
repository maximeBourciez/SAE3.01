Description détaillée

Classe représentant un contenu multimédia.

Cette classe définit la structure d'un contenu (film, série, etc.) avec toutes ses propriétés et méthodes d'accès. Elle gère les informations comme le titre, la date, les descriptions et les liens vers les affiches.
